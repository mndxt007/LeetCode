# LeetCode Solutions

My attempts at solving LeetCode problems using C# (.NET file-based apps).

## Running

Each `.cs` file is a standalone [file-based app](https://learn.microsoft.com/en-us/dotnet/core/sdk/file-based-apps) that runs with:

```bash
dotnet run <file>.cs
```

Requires .NET 10 SDK or later.
